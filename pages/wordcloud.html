How to plot wordcloud in python¶

Word clouds are extremely intuitive way to show word counts in a visually appealing manner. We will use the package wordcloud for this.

Install wordcloud using pip and lets show couple of ways of using it.

Generate from word frequencies¶

In case you already have a dict of word frequencies, you can generate it by method below:

  • Lets generate word frequency dict for sources
In [7]:
source_freq_df = pd.DataFrame(df["source"].value_counts().reset_index())
source_freq_df.columns = ["source", "frequency"]
source_freq_df.head()

In [8]:
source_freq_dict = dict(zip(source_freq_df["source"],source_freq_df["frequency"]))
In [9]:
wordcloud = WordCloud().generate_from_frequencies(frequencies=source_freq_dict)
# Generate plot
plt.imshow(wordcloud)
plt.axis("off")
